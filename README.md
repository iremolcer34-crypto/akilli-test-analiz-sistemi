# Akıllı Test Analiz Sistemi

Java ve SQLite kullanılarak geliştirilen Android tabanlı test ve kitap analiz sistemi.

## Kullanılan Teknolojiler
- Java
- SQLite
- Android Studio
- RecyclerView
- CardView
- DAO Architecture

## Özellikler
- Kitap ve test kayıt yönetimi
- Dinamik listeleme yapısı
- SQLite veritabanı entegrasyonu
- Kullanıcı dostu mobil arayüz
- Analiz ve takip sistemi

## Proje Durumu
Development in Progress

## Amaç
Mobil ortamda kullanıcıların test, kitap ve çalışma süreçlerini daha düzenli takip edebilmesini sağlamak.
